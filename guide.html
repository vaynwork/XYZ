How to use zoon

This is a guide on how to use [zoon]

zdata.json

This file is placed at the root directory. It functions as a configuration file to change how [zoon] works with your project.

CSS Variables

[zoon] makes use of CSS variables for easily customising your project

Variable Naming Scheme

To keep a level of consistency between themes, all CSS variables should follow a set naming scheme

Default Variables

Alter the default CSS variables to customise the look of your website
