Intro

zoon gives you access to new powerful custom elements to help you build a living website with boosted functionality

Single Page Application

With the use of zoon's custom elements you can build a great single page application

HTML files can be dynamically inserted into the page and elements are updated as the user interacts with your project

zoon Methodology

zoon works by properly seperating each part of your app to make development as intuitive as possible. Instead of a static unchanging site, zoon projects are built using responsive, living elements.
